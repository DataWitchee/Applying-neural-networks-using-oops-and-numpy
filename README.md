This project implements a fully functional Neural Network from scratch using only **NumPy** and **Python Object-Oriented Programming**, without relying on any deep learning frameworks like TensorFlow or PyTorch.

The goal of this project is to deeply understand the mathematics behind neural networks — including forward propagation, backpropagation, and gradient descent.

### 1. Forward Propagation
- Matrix multiplication of inputs and weights
- Addition of biases
- Passing through activation functions

### 2. Backpropagation
- Derivatives of loss with respect to outputs
- Chain rule for gradient computation
- Gradients flowing layer by layer

### 3. Optimization
- Gradient Descent for updating weights and biases

and at last training it on XOR..
